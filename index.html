🎬 TMDB API 키
영화 검색을 사용하려면 무료 TMDB API 키가 필요합니다.
1Safari에서 themoviedb.org 접속 → 무료 가입
2우측 상단 프로필 → 설정 → API 메뉴로 이동
3API 키 (v3 auth) 항목의 키 복사
4아래 입력창에 붙여넣기 후 저장
→ TMDB API 설정 페이지 열기
🗒️ 앱 아이콘 업데이트 방법
홈화면 아이콘 디자인이 변경되었을 때:
① 홈화면에서 CineDiary 아이콘 꾹 누르기 → 북마크 삭제
② Safari에서 파일 다시 열기
③ 하단 공유버튼(□↑) → 홈 화면에 추가
📤 데이터 내보내기 (CSV)
내 모든 관람 기록을 CSV 파일로 저장합니다. 엑셀, 구글 스프레드시트에서 열 수 있습니다.
☁️ 구글 시트 클라우드 동기화
구글 시트가 설정되면 폰을 바꿔도, 앱을 삭제해도 데이터가 구글 서버에 안전하게 보관됩니다.
CONFIG의 SHEET_ID, CLIENT_EMAIL, PRIVATE_KEY를 설정하면 자동 동기화됩니다.
